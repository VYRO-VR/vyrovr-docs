The **premium strap** is an optional upgrade over the comfort strap, offered as part of the Premium Strap Bundle. It uses a higher-grade band, different fasteners, and a more secure tracker tray.

## What's different from the comfort strap

- Higher-quality strap material (less stretch, more durable in the long term)
- Refined tracker tray fit
- Cleaner routing without external velcro tails

## Where each strap goes

Same as the comfort strap — see [Wearing Trackers](/straps/wearing-trackers/) for body-part positions and tracker orientation.
