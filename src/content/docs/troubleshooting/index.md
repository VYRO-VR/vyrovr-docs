If something isn't working, find the closest symptom below and follow the link. If your issue doesn't match anything here, jump into [Discord](https://discord.gg/vyrovr) — fastest path to help, and we'll add the answer back to this site.

## Pairing / connection

- **A tracker isn't appearing in the SlimeVR window** → [Tracker Won't Pair](/troubleshooting/tracker-not-pairing/)
- **The receiver doesn't appear plugged in** → [Receiver Not Detected](/troubleshooting/receiver-not-detected/)
- **Trackers connect but drop out during play** → [Range & Placement](/receiver/range-and-placement/)

## Tracking quality

- **My avatar is drifting / facing the wrong way** → [Drift & Resets](/troubleshooting/drift-and-resets/)
- **Knees bending the wrong way / foot inside the floor** → re-run [Mounting Calibration](/slimevr-server/mounting-calibration/) and [Body Proportions](/slimevr-server/body-proportions/)
- **Trackers feel slow / laggy** → [Range & Placement](/receiver/range-and-placement/) (interference)

## SteamVR

- **SteamVR doesn't see my trackers** → [Trackers Not Showing in SteamVR](/troubleshooting/steamvr-not-showing/)
- **Trackers appear but are frozen** → make sure SlimeVR Server is running before SteamVR

## Power / charging

- **A tracker won't power on** → [Charging Issues](/troubleshooting/charging-issues/)
- **Battery only lasts a few hours** → [Short Battery Life](/troubleshooting/short-battery-life/)
- **Tracker getting warm** → normal during charge; concerning if hot during use

## Software

- **SlimeVR Server won't launch** → reinstall from [slimevr.dev](https://slimevr.dev). Most install issues are missing Java or a blocked firewall.
- **SlimeVR Server window is empty / black** → restart the server. If persistent, reinstall.
